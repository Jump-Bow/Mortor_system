# FEM 設備保養管理系統 - Web UI 使用指南

## 🎨 Web UI 功能概覽

本系統提供完整的 **AdminLTE** 風格管理後台介面，支援所有核心功能操作。

### ✅ 已實作的頁面

#### 1. 認證與登入
- **登入頁面** (`/login`)
  - 本地帳號密碼登入
  - Azure AD 登入按鈕（待實作）
  - 記住我功能
  - 美觀的登入介面

#### 2. 儀表板 (`/dashboard`)
- **異常追蹤統計卡片**
  - 今日異常項目
  - 今日注意項目
  - 累積異常未結案
  - 累積注意未結案

- **巡檢作業統計**
  - 未派工任務
  - 未完成任務
  - 執行中任務
  - 已完成任務

- **視覺化圖表**
  - 任務狀態分布圓餅圖
  - 異常趨勢折線圖（最近7天）

- **最近巡檢紀錄表格**
  - 即時數據更新
  - 完成率進度條
  - 異常標記

#### 3. 巡檢管理

##### 3.1 巡檢紀錄查詢 (`/inspection/records`)
- **多條件篩選**
  - 日期範圍篩選
  - 組織篩選（支援 Select2）
  - 設備篩選
  - 任務編號搜尋
  - 馬達類別篩選
  - 保養週期篩選
  - 狀態篩選
  - 異常篩選

- **資料展示**
  - DataTables 分頁表格
  - 中文語系支持
  - 狀態 Badge 標籤
  - 完成率進度條

- **詳細資訊查看**
  - 任務基本資訊
  - 管制點摺疊展開
  - 檢查項目詳細列表
  - 檢查結果與異常原因

- **功能按鈕**
  - 查詢按鈕
  - 重置按鈕
  - 匯出按鈕（待實作）

##### 3.2 異常追蹤管理 (`/inspection/abnormal-tracking`)
- **篩選功能**
  - 異常類型（異常/注意）
  - 案件狀態（未結案/已結案）
  - 組織篩選
  - 設備篩選
  - 馬達類別篩選
  - 保養週期篩選
  - 日期範圍

- **追蹤列表**
  - 異常詳細資訊
  - 負責人顯示
  - 發現時間
  - 異常原因摘要

- **詳情彈窗**
  - 完整異常資訊
  - 檢查相關資料
  - 結案功能（待實作）

##### 3.3 抄表趨勢圖(設備) (`/inspection/trend`)
- **左側設備選擇面板**
  - 依 `TOrganization` 組織分群，可展開/收合
  - 即時搜尋設備名稱
  - 每組織顯示設備數量徽章

- **日期區間選擇**
  - 自訂開始/結束日期
  - 快捷鍵：近3個月 / 近6個月 / 近1年

- **雙趨勢圖**（依 `unit` 欄位自動分類）
  - **振動趨勢圖**（`unit=mm/s`）：MIH/MIV/MIA/MOH/MOV 振動量測
  - **溫度趨勢圖**（`unit=℃`）：前軸承/馬達本體/後軸承溫度
  - 每個 max_v 各顯示一條紅色上限參考線
  - 圖例顯示項目名稱及上限值

- **量測明細記錄表**
  - 日期、項目、量測值、單位、上限、狀態
  - 異常列以紅色標記

##### 3.4 同性質設備趨勢比較 (`/inspection/comparison`)
- **左側檢查項目樹**
  - 依量測類別分類：馬達-振動 / 馬達-溫度
  - 去重顯示唯一項目名稱
  - 可搜尋、展開/收合

- **日期區間選擇**
  - 自訂日期 + 快捷鍵（近3月/6月/1年）

- **設備勾選表格**
  - 顯示所有設備的組織/代號/名稱/上下限
  - 全選/取消全選
  - **「顯示趨勢圖」按鈕**（右上角）觸發比較圖

- **多設備趨勢比較圖**
  - 每台設備一條顏色線（最多10種顏色）
  - 紅色上限參考線
  - 點擊後頁面自動滾動到圖表位置

#### 4. 任務管理

##### 4.1 任務列表 (`/task/list`)
- **任務篩選**
  - 狀態篩選
  - 日期範圍

- **任務資訊**
  - 任務編號
  - 路線資訊
  - 指派人員
  - 完成率
  - 開始/結束時間

- **操作功能**
  - 查看詳情
  - 編輯任務（待實作）
  - 新增任務（待實作）

##### 4.2 巡檢進度查詢 (`/inspection/progress`)
- **進度查詢**
  - 組織篩選
  - 馬達類別
  - 保養週期
  - 狀態篩選
  - 日期範圍

- **工單資訊**
  - 工單基本資料
  - 設備數量統計
  - 完成率展示
  - 工單類型標籤

#### 5. 組織管理

##### 5.1 組織架構 (`/organization/tree`)
- **樹狀結構展示**
  - jsTree 互動式組織樹
  - 點擊節點顯示詳情
  - 多層級展開/收合

- **組織資訊**
  - 組織代碼與名稱
  - 組織層級
  - 完整路徑
  - 啟用狀態

- **關聯路線**
  - 組織下的巡檢路線列表
  - 路線類型標籤
  - 路線詳情查看

## 🚀 使用方式

### 1. 啟動應用

```bash
# 確保已安裝依賴
pip install -r requirements.txt

# 初始化資料庫
python init_db.py

# 啟動應用
python run.py
```

### 2. 訪問 Web UI

開啟瀏覽器訪問：`http://localhost:5000`

### 3. 登入系統

使用預設管理員帳號：
- **使用者名稱**: `admin`
- **密碼**: `1234qwer5T`

### 4. 瀏覽功能

1. **查看儀表板統計**
   - 自動載入最新數據
   - 5分鐘自動刷新

2. **查詢巡檢紀錄**
   - 設定日期範圍
   - 選擇篩選條件
   - 點擊查詢按鈕
   - 查看詳細資訊

3. **管理異常追蹤**
   - 篩選未結案項目
   - 查看異常詳情
   - 指派負責人
   - 結案處理

4. **管理任務**
   - 查看任務列表
   - 檢查任務狀態
   - 查看完成率

5. **查詢 AIMS 工單**
   - 輸入工單號碼
   - 查看工單進度
   - 檢視設備資訊

## 🎓 開發建議

### 新增頁面
1. 在 `app/templates` 建立 HTML 檔案
2. 繼承 `layout/base.html` 模板
3. 在 `app/web_routes.py` 註冊路由
4. 實作對應的 JavaScript 邏輯

### 修改樣式
1. 編輯 `app/static/css/custom.css`
2. 使用 AdminLTE 提供的 CSS 類別
3. 保持一致的設計語言
